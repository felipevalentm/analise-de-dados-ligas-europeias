# Análise de Dados - Ligas Europeias

## 📌 Sobre o Projeto

Projeto de análise exploratória de dados aplicado ao futebol europeu, utilizando Python para tratamento, análise e extração de insights a partir de dados de jogadores e competições.

O objetivo é explorar informações estatísticas, identificar padrões e transformar dados esportivos em informações relevantes através de técnicas de análise de dados.
